# 配信レイアウトツール (Venue Layout Planner)

ゲーム大会・配信イベントの会場レイアウトと機材配線を計画するWebツール / Venue layout & cabling planner for game tournaments (pure HTML/JS, no build)

ブラウザで `index.html` を開くだけで動作します。サーバー・ビルド・外部ライブラリは一切不要です。

**▶ ブラウザでそのまま試す: https://kijitora-no-hito.github.io/eSportsEvent_FloorPlanner/**

## 主な機能

- **間取り作成** — グリッド点をクリックして多角形の部屋を作図
- **機材配置** — 什器（机・椅子・電源/LANコンセント）と機器（PC・モニタ・ゲーム機など17種）をドラッグ&ドロップで配置。移動・90°回転・サイズ変更・名前変更に対応
- **ケーブル配線** — 機器のポート間をケーブルで接続。ポート種別（HDMI / USB / LAN / 電源）の**整合チェック**と IN/OUT の**方向チェック**付きで、物理的にありえない配線を防止
- **ケーブル長の算出** — レイアウト上の距離から必要なケーブルの最小長さを自動計算・表示（1グリッド = 0.25m 等、ズームに応じた自動スケール）
- **必要機材チェックリスト** — 配置した機材とケーブルを数量・長さ付きで自動集計。チェック状態も保存され、テキストコピーで買い出しリストとして共有可能
- **電源未接続の警告** — 電源が来ていない機器に ⚡ バッジを表示（USB給電にも対応）
- **カスタム機器の登録** — 名前・色・ポート構成を自由に定義。既存機器の複製編集も可能
- **保存** — localStorage への自動保存 + JSONファイルのエクスポート/インポート

## 使い方

1. このリポジトリをクローンまたはダウンロード
2. `index.html` をブラウザ（Chrome / Edge / Firefox）で開く

```
git clone https://github.com/kijitora-no-hito/eSportsEvent_FloorPlanner.git
cd eSportsEvent_FloorPlanner
# index.html をダブルクリックで開くだけ
```

### 基本操作

| モード | キー | 操作 |
|---|---|---|
| 間取り作成 | `1` | グリッド点をクリックして頂点追加 → 最初の点を再クリック / `Enter` / ダブルクリックで確定 |
| 配置 | `2` | パレットから選択してクリック配置。ドラッグ移動、`R` または丸ハンドルで回転、右下ハンドルでサイズ変更、`Delete` で削除、ダブルクリックで名前変更 |
| 接続 | `3` | パレットでケーブルを選択 → 緑枠のポートを2つクリックで接続。ケーブルをクリック → `Delete` で削除 |

### 画面操作

| 操作 | 方法 |
|---|---|
| ズーム | マウスホイール（カーソル位置基準）または `＋` / `－` ボタン、`100%` で等倍 |
| パン | ホイール押し込みドラッグ、または `Space` + 左ドラッグ |
| キャンセル | `Esc`（作図・配置・接続の中断、選択解除） |

拡大するとグリッドが自動で細かくなり（最小 0.125m）、ポートや文字は画面上のサイズを保つため、詳細な配線設計がしやすくなっています。現在のグリッド寸法は画面右下に表示されます。

### 表示の切り替え

パレットの「什器 / 機器 / ケーブル」見出し右のチェックボックスで、分類ごとに表示を半透過に切り替えられます（電源系統だけ確認したいときなどに便利）。

## 技術構成

- 素の HTML + CSS + JavaScript（ES5、フレームワーク・ビルド不要）
- 描画は SVG（`viewBox` によるズーム/パン、`vector-effect` による線幅固定）
- `file://` で直接開けるよう ES modules は使わず、classic script + 名前空間パターンで構成

```
├── index.html
├── css/style.css
└── js/
    ├── constants.js    # ポート・ケーブル種別と互換ルール
    ├── templates.js    # 機器テンプレート定義
    ├── geometry.js     # スナップ・ポート座標・ケーブル長計算
    ├── state.js        # 状態管理・保存
    ├── render.js       # SVG描画
    ├── palette.js      # パレットUI
    ├── checklist.js    # 必要機材チェックリスト
    ├── interactions.js # マウス/キーボード操作
    └── main.js         # 初期化・ズーム/パン
```
